# ClearChat
A simple spigot plugin to clear your chat! Featuring configurable messages.

## Commands
   - /clearchat
      - /cc
      
   - /ccmsgnoperm
      - /ccmnp
      
   - /ccmsgcleared
      - /ccmc
   
## Permissions
   - clearchat.clear
   - clearchat.config
