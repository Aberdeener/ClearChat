# ClearChat
A simple spigot plugin to clear your chat! 
Now featuring configurable messages.

Check it out @ https://www.spigotmc.org/resources/clearchat.66886/

## Commands
   - /clearchat
      - /cc
      
   - /ccmsgnoperm
      - /ccmnp
      
   - /ccmsgcleared
      - /ccmc
   
## Permissions
   - clearchat.clear
   - clearchat.config
